# CSE322-ns3

This repo contains the source code of the attempted implementation of RFC 9840
rLEDBAT: Receiver-Driven Low Extra Delay Background Transport for TCP


Newly added files in src/internet/model:
tcp-rledbat.h
tcp-rledbat.cc


The scratch direcotry contains the code for experimentation, graphs and the reports.
There are two reports. One covers the wired and the other covers the wireless experimentation results.
